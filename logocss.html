Pembelajaran CSS - Kelas X

1. Cara Menambahkan CSS ke HTML

Ada 3 cara untuk menerapkan CSS:

  • Inline CSS: Ditulis langsung di dalam tag HTML menggunakan atribut style
    Contoh:
    <p style="color: red; font-size: 16px;">Teks merah</p>
  • Internal CSS: Ditulis di dalam tag <style> di bagian <head> HTML
    Contoh:
    <head>
    <style>
        p { color: blue; }
    </style>
</head>
  • External CSS: Ditulis di file terpisah dengan ekstensi .css, kemudian dihubungkan ke HTML menggunakan <link>
    Contoh:
    <head>
    <link rel="stylesheet" href="style.css">
</head>

2. Sintaks Dasar CSS

Setiap aturan CSS terdiri dari selector dan deklarasi block:

selector {
    properti: nilai;
    properti-lain: nilai-lain;
}
  • Selector: Menentukan elemen HTML yang akan diatur (misal: p, .kelas, #id)
  • Properti: Apa yang akan diubah (misal: color, font-size)
  • Nilai: Pengaturan untuk properti tersebut (misal: red, 18px)

3. Jenis Selector Umum

  • Element Selector: Memilih elemen berdasarkan nama tag
    Contoh: h1 { text-align: center; } (semua tag <h1> akan rata tengah)
  • Class Selector: Memilih elemen dengan atribut class tertentu (dimulai dengan titik .)
    Contoh: .teks-kuning { color: yellow; } (elemen dengan class="teks-kuning")
  • ID Selector: Memilih elemen dengan atribut id tertentu (dimulai dengan pagar #, harus unik per halaman)
    Contoh: #header-utama { background: gray; } (elemen dengan id="header-utama")
  • Universal Selector: Memilih semua elemen (gunakan tanda bintang *)
    Contoh: * { margin: 0; padding: 0; } (menghapus margin/padding default)
  • Group Selector: Memilih beberapa selector sekaligus (pisahkan dengan koma)
    Contoh: h1, h2, h3 { font-family: Arial; }

4. Properti CSS Dasar

A. Pengaturan Teks

  • color: Warna teks (misal: red, #ff0000, rgb(255,0,0))
  • font-family: Jenis font (misal: Arial, 'Segoe UI', sans-serif)
  • font-size: Ukuran teks (misal: 16px, 1.2rem, 120%)
  • font-weight: Ketebalan teks (misal: bold, 400 = normal, 700 = tebal)
  • text-align: Penjajaran teks (misal: left, center, right)
  • text-decoration: Dekorasi teks (misal: none = hilangkan garis bawah link, underline, line-through)

B. Pengaturan Latar Belakang

  • background-color: Warna latar belakang
  • background-image: Gambar latar belakang (misal: url("gambar.jpg"))
  • background-repeat: Pengulangan gambar latar (misal: no-repeat, repeat-x)
  • background-size: Ukuran gambar latar (misal: cover, contain, 50%)

C. Pengaturan Box Model (Model Kotak)

Setiap elemen HTML dianggap sebagai kotak, terdiri dari:

  • margin: Ruang di luar batas elemen
  • border: Garis batas di sekitar elemen
  • padding: Ruang di dalam batas elemen
  • width/height: Lebar/tinggi konten elemen

Contoh:

.kotak {
    width: 200px;
    height: 100px;
    padding: 15px;
    border: 2px solid black;
    margin: 10px;
}

D. Pengaturan Tata Letak

  • display: Menentukan cara elemen ditampilkan
    Misal: block (menempati baris penuh), inline (berbaris dengan elemen lain), inline-block, none (sembunyikan elemen)
  • float: Menggeser elemen ke kiri/kanan (misal: left, right)
  • position: Pengaturan posisi elemen
    Misal: static (default), relative (posisi relatif dari tempatnya), absolute (posisi relatif dari induk yang memiliki posisi), fixed (tetap di tempat meskipun discroll)
