Pembelajaran JavaScript - Kelas X

1. Cara Menambahkan JavaScript ke HTML

Ada 3 cara untuk menjalankan kode JavaScript:

  • Inline JS: Ditulis di dalam atribut HTML seperti onclick
    Contoh:
    <button onclick="alert('Halo!')">Klik Saya</button>
  • Internal JS: Ditulis di dalam tag <script> di bagian <head> atau <body> HTML
    Contoh:
    <body>
    <script>
        console.log("Halo dari JavaScript!");
    </script>
</body>
  • External JS: Ditulis di file terpisah dengan ekstensi .js, kemudian dihubungkan ke HTML
    Contoh:
    <body>
    <script src="script.js"></script>
</body>
    Saran: Letakkan tag <script> di akhir <body> agar konten HTML dimuat terlebih dahulu.

2. Sintaks Dasar

  • Komentar: Untuk catatan atau menonaktifkan kode
    Contoh:
    // Komentar satu baris

/* 
Komentar 
banyak baris 
*/
  • Pernyataan: Setiap perintah diakhiri dengan titik koma (;) (opsional tapi disarankan)
    Contoh: let nama = "Sharon";
  • Pemisah Baris: JavaScript akan membaca baris baru sebagai pemisah pernyataan jika tidak ada titik koma, tapi bisa menyebabkan error jika tidak tepat.

3. Variabel dan Tipe Data

Variabel digunakan untuk menyimpan nilai. Ada 3 cara mendeklarasikan variabel:

  • let: Nilainya bisa diubah nanti (bersifat blok-level)
  • const: Nilainya tidak bisa diubah setelah dideklarasikan (konstanta)
  • var: Cara lama (tidak disarankan karena cakupannya lebih luas)

Tipe Data Utama:

  • String: Teks, ditulis dengan kutip tunggal ('...') atau ganda ("...")
    Contoh: let pesan = "Halo Dunia!";
  • Number: Angka (bilangan bulat atau desimal)
    Contoh: let umur = 25; let tinggi = 1.65;
  • Boolean: Nilai benar (true) atau salah (false)
    Contoh: let isOnline = true;
  • Null: Nilai kosong yang ditentukan secara sengaja
  • Undefined: Nilai yang belum ditetapkan
  • Object: Koleksi pasangan kunci-nilai (misal: objek pengguna)
    Contoh:
    const pengguna = {
    nama: "Sharon",
    pekerjaan: "Developer",
    umur: 25
};
