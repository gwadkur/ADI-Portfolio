Pembelajaran HTML - Kelas X

1. Elemen Teks Dasar

Digunakan untuk menampilkan dan memformat teks:

  • <h1> hingga <h6>: Judul dengan tingkat kepentingan berbeda (h1 paling besar, h6 paling kecil)
  • <p>: Paragraf teks
  • <br>: Pemisah baris baru
  • <hr>: Garis horizontal pemisah
  • <strong> atau <b>: Teks tebal (strong lebih menekankan makna secara semantik)
  • <em> atau <i>: Teks miring (em lebih menekankan makna secara semantik)
  • <u>: Teks bergaris bawah
  • <blockquote>: Kutipan panjang
  • <code>: Teks kode program

2. Elemen Penyusun Struktur

Digunakan untuk mengatur tata letak konten:

  • <div>: Wadah umum untuk mengelompokkan elemen (blok-level)
  • <span>: Wadah umum untuk mengelompokkan teks atau elemen kecil (inline)
  • <header>: Bagian kepala halaman atau bagian
  • <nav>: Bagian navigasi
  • <main>: Konten utama halaman
  • <section>: Bagian khusus dalam halaman
  • <article>: Konten yang mandiri (misal: artikel blog)
  • <footer>: Bagian kaki halaman atau bagian

3. Elemen Media

Digunakan untuk menampilkan konten multimedia:

  • <img src="url-gambar" alt="deskripsi-gambar">: Menampilkan gambar (atribut alt penting untuk aksesibilitas)
  • <audio src="url-audio" controls>: Memutar file audio (atribut controls menampilkan tombol kontrol)
  • <video src="url-video" controls width="lebar">: Memutar file video

4. Elemen Link dan Navigasi

  • <a href="url-tujuan" target="_blank">Teks Link</a>: Membuat tautan (atribut target="_blank" membuka di tab baru)
  • <a href="#id-elemen">Scroll ke Bagian Ini</a>: Tautan internal ke elemen dengan ID tertentu
